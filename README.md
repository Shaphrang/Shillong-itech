# Shillong ITech Website

## Run locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Update phone/email
Edit `src/data/site.ts` and update `phone`, `whatsapp`, and `email`.

## Deploy to Vercel
1. Push to GitHub.
2. Import project in Vercel.
3. Use defaults and deploy.

## Update services/projects
- Services: `src/data/services.ts`
- Projects: `src/data/projects.ts`

## SEO checklist after deployment
- Verify sitemap: `/sitemap.xml`
- Verify robots: `/robots.txt`
- Test mobile responsiveness
- Test Open Graph in social preview tools
- Submit domain in Google Search Console
